# Attendance-system


A real-time face detection and attendance logging system built using **Python, OpenCV, and MediaPipe**. The application detects faces through a webcam feed and automatically records presence timestamps in a CSV file.

---

## 🚀 Features

* Real-time webcam face detection
* MediaPipe Face Detection integration
* Automatic attendance logging
* Timestamp recording in CSV format
* Lightweight and fast performance
* Works on Python 3.12

---

## 🛠️ Technologies Used

* Python
* OpenCV
* MediaPipe
* CSV File Handling

## ⚙️ Installation

### Clone Repository

```bash
git clone https://github.com/yourusername/Attendance_System.git
```

### Navigate to Project Directory

```bash
cd Attendance_System
```

### Install Dependencies

```bash
pip install opencv-python mediapipe
```

### Run the Application

```bash
python main.py
```

---

## 🧠 How It Works

1. Opens webcam feed.
2. Detects faces using MediaPipe Face Detection.
3. Draws bounding boxes around detected faces.
4. Records presence timestamps in a CSV file.
5. Displays real-time detection status on screen.
